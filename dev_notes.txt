Compile Web Modern:
- cd traccar-web

- cd ../../

Deploy Steps:
- Build Server
- Build Modern Web App
- cd setup/
- sh package.sh 6.2 linux-64
- On both Prod servers:
    - rm /home/bunny/build/*
- Transfer build to server:
    - rsync -a --progress traccar-linux-64-6.2.zip nconnect_gps:/home/bunny/build/
    - rsync -a --progress traccar-linux-64-6.2.zip nspira_gps:/home/bunny/build/
    - Repeat above step for nspira_gps too.
- ssh into each server:
    - mkdir ./backup/<date>/
    - cp /opt/traccar/conf/traccar.xml ./backup/<date>/
    - sudo service traccar stop && cd build/ && unzip traccar-linux-*.zip && sudo ./traccar.run
    - Check if all okay:
        - sudo nano /opt/traccar/conf/traccar.xml
    - cd /opt/traccar/ && ls -lt
    - sudo service traccar start && tail -f /opt/traccar/logs/tracker-server.log
- Check respective URLs, Login + Check console.

Transfer code command example:
- rsync -a --progress traccar-linux-64-5.6.zip nconnect_gps:/home/bunny/build/

Check server logs:
- tail -f /opt/traccar/logs/tracker-server.log
- grep 866897057683700 tracker-server.log
- For grep, add "-B 10" option to show +10 lines before the grep match line.

Check traccar service logs:
- sudo journalctl -u traccar.service --since "2022-07-11" --no-pager
- sudo journalctl -u traccar.service -f --since "1 hour ago"
- sudo journalctl -u traccar.service -f --since "1 second ago"
- sudo journalctl -u traccar.service --since "2023-04-15 02:28:00" --until "2023-04-15 02:30:00" --no-pager
