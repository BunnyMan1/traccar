Compile Server:
- gradle
- ./gradlew assemble

Compile Web Modern:
- cd traccar-web/modern
- npm i
- npm run build

Deploy:
- Server: Upload target/lib and target/tracker-server.jar to the server at /opt/traccar
- Web modern: Upload /traccar-web/modern/build folder to the server at /opt/traccar/web/modern
- Steps:
    - Compare all files using tree with new source installation.
    - Stay at /home/bunny
    - sudo su
    - cp -r /opt/traccar/lib backup/lib_v_5_2/
    - cp -r /opt/traccar/tracker-server.jar backup/tracker-server_v_5_2.jar
    - service traccar stop
    - rm -rf /opt/traccar/lib && mv build/lib /opt/traccar/
    - mv build/tracker-server.jar /opt/traccar/
    - mv build/build /opt/taccar/modern
    (Check if conf/default.xml needs to be updated from new source installation)
    - service traccar start

Transfer code command example:
- rsync -a --progress temp_upload/ nspira_gps:/home/bunny/build/
- rsync -a build nspira_gps:/home/bunny/build/
